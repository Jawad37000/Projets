# Traffic Flow

Ce projet calcule **le débit horaire maximal de véhicules** entre un point d'entrée (E) et un point de sortie (S) dans un réseau routier dirigé.

## Installation

```bash
uv venv
source .venv/bin/activate  
uv pip install -e .
```

## Exécution

```bash
python demo.py
```

## Tests

```bash
python -m unittest discover tests
```
